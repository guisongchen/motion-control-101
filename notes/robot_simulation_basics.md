# 机器人仿真基础概念

## 1. 初始落地（Landing Phase）

### 什么是初始落地
在仿真开始时，让机器人从空中自由落体到地面，经历一个无控制的稳定过程：

```python
# 让机器人稳定落地 (不控制，自由落体)
for _ in range(100):
    p.stepSimulation()
```

### 为什么要这样做

| 目的 | 说明 |
|------|------|
| 避免空中误控 | 如果刚加载就启动PD控制，机器人还在自由落体，控制逻辑会基于错误状态计算 |
| 找到自然平衡点 | 初始设定的关节角度只是"姿势"，落地后关节会在重力作用下自然压缩，找到真实的站立状态 |
| 渐进建立接触 | 脚与地面的接触是渐进建立的，而非硬碰硬，数值更稳定 |

### 落地后的状态
- 代码示例中：高度 0.889m，倾斜 4.6°
- 说明落地后躯干已有轻微前倾，不是完全直立

## 2. 为什么从空中加载机器人

### 基座（Base Link）的概念

```
根部 = URDF基座 (base_link) = 躯干/骨盆
```

- **基座**是整个机器人结构的根节点
- 所有其他部件（腿、手臂）都通过关节连接到这个根上
- 对于人形机器人，基座在**腰部/骨盆**，不是脚底

### 加载位置的含义

```python
p.loadURDF("humanoid.urdf", [0, 0, 1.2])  # 基座在高度 1.2m
```

| 加载高度 | 结果 |
|---------|------|
| 0m | 腰部埋在地下，腿部穿透地面 ❌ |
| 0.9m | 腰部贴地，腿被压扁 ❌ |
| 1.2m | 腰部悬空，腿自然垂下，落地后脚正好接触地面 ✓ |

### 避免碰撞穿透（Penetration）
如果基座直接与地面重叠：
- 机器人可能被"弹飞"
- 接触力计算错误
- 关节进入奇怪的状态

## 3. URDF 关节基础

### 关节类型（Joint Type）

| 类型 | 说明 | 示例 |
|------|------|------|
| `revolute` | 旋转关节，有角度限制 | 膝、肘 |
| `continuous` | 连续旋转，无角度限制 | 轮子 |
| `prismatic` | 滑动关节 | Cart-Pole 的小车 |
| `fixed` | 固定连接 | 传感器安装在躯干 |

### 关节轴（Axis）

```
axis = (1, 0, 0)  # 绕 x 轴旋转 → 俯仰（前后倾斜）
axis = (0, 1, 0)  # 绕 y 轴旋转 → 横滚（左右摇摆）
axis = (0, 0, 1)  # 绕 z 轴旋转 → 偏航（左右转身）
```

例如人形机器人的脚踝：
- 通常只有 x 轴（俯仰），不能左右摇摆
- 髋关节有多个关节（x,y,z）来实现球关节效果

### 父子关系（Parent/Child）

```
基座(torso)
    ├── 髋(hip) → 大腿
    │       └── 膝(knee) → 小腿
    │               └── 踝(ankle) → 脚
    ├── 肩(shoulder) → 上臂
    │       └── 肘(elbow) → 前臂
```

形成树状结构，从基座向外延伸。

### 关节限制

- **角度限制**：如膝盖 0~150°，不能反向弯曲
- **力矩限制**：代码中 `max force` 防止数值溢出

## 4. 控制模式对比

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| `POSITION_CONTROL` | 设定目标角度，电机自动计算力矩 | 固定姿态（如髋膝固定） |
| `TORQUE_CONTROL` | 直接输出力矩 | 实时平衡控制 |
| `VELOCITY_CONTROL` | 设定目标速度 | 释放关节（force=0） |

为什么平衡控制必须用扭矩模式？
- 需要根据倾斜角度和角速度**实时计算**纠正力
- 位置模式是"追踪预设轨迹"，不适合应对扰动

## 5. 调试技巧

### 打印关节列表

```python
for i in range(p.getNumJoints(robot)):
    info = p.getJointInfo(robot, i)
    name = info[1].decode('utf-8')
    print(f"{i}: {name}")
```

### 常见错误

| 错误 | 原因 |
|------|------|
| `Joint index out-of-range` | 关节索引写错，需用上面代码确认 |
| 机器人被弹飞 | 初始位置与地面重叠 |
| 关节不响应控制 | 没切换到正确的控制模式 |
